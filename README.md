# apex-movies
A prototype React app that fetches TMDb, IMDB and Wikipedia data available for movies.

**Libraries used:**
- Typescript
- React
- GraphQL
- Axios
- Material UI
- Jotai
- SCSS
- Jest

**Tasks complete:**
- [x] Working prototype
- [x] Loading spinner
- [x] Related search
- [x] Material UI
- [ ] Tests

## Installation
```

git clone https://github.com/ppajer/apex-movies.git
cd apex-movies
npm install
npm start

```
